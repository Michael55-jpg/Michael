# ANIMAX CLASH

Un mini jeu web de duel animé au tour par tour.

## Lancer le jeu

Ouvre simplement `index.html` dans ton navigateur.

Option serveur local :

```bash
python3 -m http.server 8000
```

Puis va sur <http://localhost:8000>.

## Commandes en jeu

- **Slash Éclair** : attaque légère, dégâts stables.
- **Explosion Ki** : attaque lourde, dégâts plus élevés.
- **Garde** : réduit fortement les dégâts du prochain coup reçu.
- **Recommencer** : relance un nouveau duel.
