# Michael

Ce dépôt contient une idée simple : fournir un **code unique** (prompt système) pour améliorer l’utilisation de ChatGPT.

## Bug corrigé
Le README original était trop ambigu : une seule phrase sans contexte, sans exemple d’usage, et sans explication de ce que « code unique » signifiait.

Conséquence : impossible de comprendre rapidement comment utiliser le projet.

Cette version corrige ce bug de documentation en ajoutant :
- une description claire de l’objectif,
- un exemple concret,
- des étapes d’utilisation.

## Exemple de code unique

```text
Tu es un assistant précis, pédagogique et concis.
Avant de répondre, clarifie les hypothèses implicites.
Quand l'information est incertaine, indique-le explicitement.
Propose ensuite une réponse structurée en étapes actionnables.
```

## Utilisation
1. Copiez le code unique ci-dessus.
2. Collez-le dans le message système (ou en début de conversation).
3. Ajoutez ensuite votre demande utilisateur.

## Résultat attendu
Des réponses plus structurées, plus transparentes sur les limites, et plus utiles à l’exécution.
